# Frontend Slides Skill Overview

This skill creates zero-dependency, animation-rich HTML presentations from scratch or by converting PowerPoint files.

## What It Does

- **New Presentations**: Creates slides from scratch based on user content
- **PPT Conversion**: Converts .ppt/.pptx files to web presentations
- **Visual Style Discovery**: Shows style previews instead of asking for abstract preferences
- **Viewport-Fitted**: Every slide fits exactly in the viewport (no scrolling)

## Key Files

| File | Purpose |
|------|---------|
| `SKILL.md` | Main skill instructions for Claude Code |
| `STYLE_PRESETS.md` | 12 curated visual styles with CSS snippets |

## Available Styles

**Dark Themes:** Bold Signal, Electric Studio, Creative Voltage, Dark Botanical, Neon Cyber, Terminal Green

**Light Themes:** Notebook Tabs, Pastel Geometry, Split Pastel, Vintage Editorial, Swiss Modern, Paper & Ink

## Philosophy

1. Zero dependencies (single HTML files)
2. Show, don't tell (visual previews)
3. Distinctive design (no generic AI aesthetics)
4. Production quality (well-commented, accessible)

## Integration

This skill can be used with:
- Claude Code: Copy files to `~/.claude/skills/frontend-slides/`
- GitHub Copilot CLI: Use commands from `.copilot/commands/`
